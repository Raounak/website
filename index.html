About Me

I'm a Postdoc researcher at CEA LIST. I'm attached to the BINSEC team from Software Safety & Security Lab.I am currently working on software vulnerability assessment and prioritization using deep learning techniques.

During my PHD, I worked on software vulnerability detection using machine learning techniques in order to automatically identify security vulnerabilities in the source code level. This work was done with Salah Sadou and Isabelle Borne from UBS (Univeristy South Brittany) in Vannes.

Despite the topics I have worked on, I'm very interested in several topics related to software security using Artificial Intelligence methods such as automatic program repair.
